# Nifty Alpha

Mobile-first intraday Nifty options companion: Smart Strike selector, real-time analytics (PCR, Max Pain, OI), one-tap execution, and dark theme for trading.

## Features (PRD)

- **Smart Strike Selector** – Bullish / Bearish / Neutral / Scalping with suggested strike (e.g. Nifty ±50 for ITM, ATM for scalping).
- **Real-Time Analytics** – PCR (Put-Call Ratio) with overbought/oversold, Max Pain, multi-strike OI change highlights.
- **Execution** – Buy, Exit All, auto trailing stop-loss at 10% profit (breakeven lock), basket order placeholder.
- **Dashboard** – Live Nifty spot, analytics card, Nifty 50 sector heatmap placeholder.
- **Alerts** – OI spike and key level notifications (placeholder list + manage rules).
- **Dark theme** – Default dark mode to reduce eye strain.

## Run

```bash
flutter pub get
flutter run
```

Use **Android** or **iOS** for mobile; **Chrome** for web.
